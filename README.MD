🌱 SmartGreen — Sistema de Monitoramento de Estufas

Este repositório faz parte das soluções desenvolvidas pela SmartGreen, uma empresa dedicada à aplicação de tecnologia no agronegócio.

A proposta do projeto é realizar o monitoramento de estufas de plantação de tomate por meio de sensores de temperatura e umidade, permitindo um controle mais preciso das condições ambientais. Dessa forma, busca-se reduzir perdas causadas por variações inadequadas desses fatores, garantindo maior qualidade e produtividade da plantação.

📁 Sobre este repositório

Este repositório, em específico, é dedicado ao armazenamento dos códigos desenvolvidos na disciplina de Algoritmos. Aqui estão reunidos os arquivos de HTML e CSS, além da lógica de programação e scripts iniciais relacionados à interface e ao funcionamento do sistema de monitoramento da SmartGreen.
